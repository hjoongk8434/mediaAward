# 영화,드라마 검색 사이트 #
- TMDB(The Movie Database)를 이용하여 영화, 드라마 인기순위와 검색을 통해 TMDB에 저장된 데이터를 불러오는 사이트 입니다.

### 프로젝트 스펙  ###
- 스프링 프레임워크 4.3.30 RELEASE (Legacy)
- 자바 17
- 톰캣 9.0
- 메이븐 3
- 인텔리제이 IDE

### 사전 체크 ###
 - TMDB(The Movie Database) 사이트 접속하여 API Keys 발급.
 - 발급된 API keys를 프로젝트안에 있는 external.properties의 td.id.access값을 설정.
 - 로컬로 실행하면 localhost:8080으로 접속.

업무를 담당했던 회사의 시스템들이 스프링 프레임워크 레거시 방식으로 구성이 되어있어
레거시로 작업을 하여 새로 다시 공부하기 위해서 만든 프로젝트 입니다.
